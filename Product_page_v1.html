Material Options

There are two primary material options for the Floor Generator:

  1. MultiTexture: Allows you to use a separate texture for every board or tile.
  2. Seamless Texture: Uses a single seamless texture, with each board getting a unique random offset.

MultiTexture Workflow

The MultiTexture Node allows you to use up to 16 different textures. Simply add your textures in the Shader Editor and connect them to the inputs. This node can also be used with just RGB color inputs to create fully procedural materials without image textures.

Seamless Texture Workflow

This method uses one seamless texture. The FloorGenerator unwraps each board to a random position in the UV space, ensuring each board has a unique, non-repeating look.

Asset Browser Library

For quick access, add the FloorGenerator and materials to the Asset Browser in Blender. This package includes 56 pre-made materials to get you started:

  • 35 MultiTexture wood flooring materials
  • 2 Seamless wood flooring materials
  • 19 MultiTexture tile materials

Advanced Masking

To mask textures and materials, you can use three attributes: Bevel, Sides, and Grout. Use their factor outputs with Mix Color or Mix Shader nodes for detailed control.
